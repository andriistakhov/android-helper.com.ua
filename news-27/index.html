Из последних новостей

Предлагаем вашему вниманию очередную подборку с ссылками на новости, интересные материалы и полезные ресурсы.

Горячая семерка

Прошедшие и будущие мероприятия октября

18 октября в Киеве стартовала международная конференция IDCEE, а сегодня были названы победители конкурса стартапов. Недавно прошла Fronteers 2012 — лучшая конференция по клиентской разработке, с отчетом можно ознакомиться здесь. Также, на прошлой неделе проходила YAC, с которой уже стали доступны видео докладов. Кроме этого, намечается парочка интересных мероприятий: Apple «покажет что-то» 23-го октября, а в конце месяца Google скорее всего покажет новую версию Android

Bootstrap

Проект Bootstrap продолжает оставаться в тренде среди веб-разработчиков. Отличную подборку всякого полезного для него можно найти в этой статье. Кроме этого, желающие могут ознакомиться с туториалом по созданию адаптивной темы для WordPress с помощью Bootstrap

Internet Explorer 10

Интересная статья, где подробно можно прочесть о том, чего стоит ожидать от нового IE10. А здесь можно ознакомиться с нюансами работы Responsive Design в Snap Mode. Плюс, ожидаем предварительную верси. IE10 для Windows 7 к середине ноября

Расширяем CSS

Препроцессоры становятся все популярней. Эта статья покажет, на сколько гибкими могут стать таблицы стилей с использованием OOCSS + Sass. Здесь можно посмотреть, как с помощью несложного SCSS-миксина грузить изображения для ретина-дисплеев. Те, кто хорошо воспринимает на слух англоязычную информацию, могут открыть для себя SassCast — подкаст на тему Sass. Ну, и напоследок загляните вsassmeister — песочницу для работы с Sass/SCSS

Учебник jQuery для начинающих

Многие, кто начинал учить jQuery по русскоязычным ресурсам, не могли обойти стороной сайт Антона Шевчука, а некоторые даже начинали изучение со статей на этом сайте. Антон решился и собрал все свои статьи из серии «jQuery для начинающих» в одном учебнике, который и представил на суд общественности.

Surface на подходе

Microsoft раскрыла цены на Surface RT в официальном магазине, по итогам чего младшая модель Surface была распродана за сутки. Тем временем Microsoft запускает на ТВ новую рекламу Microsoft Surface ипервый рекламный ролик Windows 8

Google StreetView приглашает побродить в одном из их датацентров

В Google решили показать один из своих датацентров, находящийся в городе Ленуар. Теперь многие смогут убедиться, что креативные идеи в оформлении помещений айтишного гиганта используются не только в рабочих офисах и комнатах отдыха, но и в теж же датацентрах
